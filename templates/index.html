A minimal proof-of-concept cluster scorer.

To use, upload a gene expression file; each sample in the file will be assigned to a cluster.

The file must meet the following requirements:

  • The first column is gene names.
  • The rest of the columns are samples.
  • The dataset must include all genes in the scoring set, in any order.
  • The fields are separated with tabs (\t).
